Permite incluir nuestra carpeta compartida de Dropbox, Google Drive en nuestras publicaciones.

== Description ==

Para quienes es necesario vincular de alguna manera su disco en la nube con sus publicaciones para que puedan ser descargado o solamente visualizados.

Con este Plugin podras:

*   Incluir contenido de una carpeta compartida en la Nube.
*   Personalizar nombre de carpeta compartida.
*   BBCode y boton en el editor

Leer **[Mas > Descripcion](http://www.hyno.ok.pe/wp-plugins/cloud-folder-share/ "Cloud Folder Share")**

== Installation ==

1. Descomprimir el plugin a la carpeta `wp-content\plugins\`.
2. Activarlo entrando a "administracion de Plugins" (AdminPanel/Plugins).
3. Podras usar el Plugin con el BBCode o con el boton en editor de Wordpress:

PD. Habilitar la extension **php_openssl.dll** en **php.ini**.

Leer **[Mas > Instalacion](http://www.hyno.ok.pe/wp-plugins/cloud-folder-share/)**
